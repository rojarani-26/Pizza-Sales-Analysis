# 🍕 Pizza Sales Data Analysis Using SQL

## 📌 Project Overview
This project analyzes a **Pizza Sales dataset** using SQL to extract meaningful business insights. The analysis focuses on understanding **sales performance, customer purchasing behavior, and product popularity**.

Using SQL techniques such as **aggregations, grouping, subqueries, and window functions**, the project identifies important metrics like total revenue, top-selling pizzas, category performance, and monthly order trends.

The insights from this analysis can help businesses make **data-driven decisions to improve sales and customer satisfaction**.

---

## 🎯 Objectives
- Calculate the **total revenue generated from pizza sales**
- Determine the **total number of orders and pizzas sold**
- Analyze the **average order value**
- Identify the **most popular pizzas**
- Understand sales distribution by **pizza category and pizza size**
- Analyze **monthly order trends**
- Identify **high revenue generating pizzas**

---

## 🛠 Tools Used
- **SQL**
- **Microsoft SQL Server Management Studio (SSMS)**

---

## 📊 Dataset Description
The dataset contains information about pizza orders including order details, pizza types, sizes, ingredients, and pricing.

### Dataset Columns

| Column Name | Description |
|-------------|-------------|
| pizza_id | Unique identifier for each pizza record |
| order_id | Unique identifier for each order |
| pizza_name_id | Identifier for pizza type |
| quantity | Number of pizzas ordered |
| order_date | Date when the order was placed |
| order_time | Time when the order was placed |
| unit_price | Price of one pizza |
| total_price | Total price of the order |
| pizza_size | Size of the pizza (S, M, L, XL, XXL) |
| pizza_category | Category of pizza (Classic, Chicken, Veggie, Supreme) |
| pizza_ingredients | Ingredients used in the pizza |
| pizza_name | Name of the pizza |

## 🔍 Key Insights
1. The pizza business generated **$817,860.05 in total revenue** from 21,350 orders.  
2. A total of **49,574 pizzas were sold**, showing strong customer demand.  
3. The **average order value is $38.31**, indicating customers often purchase multiple pizzas per order.  
4. Order values range from **$9.75 to $83**, showing variation in customer spending behavior.  
5. The **Classic pizza category generates the highest revenue** among all categories.  
6. **Large pizzas are the most popular size**, with the highest quantity sold.  
7. **The Thai Chicken Pizza generates the highest revenue** among all pizzas.  
8. **July recorded the highest number of orders**, indicating peak demand during that month.  
9. Pizzas such as **Classic Deluxe, Barbecue Chicken, Hawaiian, Pepperoni, and Thai Chicken** are the most frequently ordered.  
10. Several pizzas generate **revenue higher than the average pizza revenue**, contributing significantly to overall sales.
